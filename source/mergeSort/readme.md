## 归并排序

* sort
* merge

### 归并

* 将俩个各自排序好的数组合并为一个排序好的数组
* 对每一半进行排序

理解：

* 宏观理解
* 微观理解

测试：

* testHelper：前一个元素永远小于等于后一个元素

### 时间复杂度

* 绘制递归树
* O(nlogn)
* 与n平方的复杂度进行对比

### 优化

* arr[mid] < arr[mid+1] 说明已经排好序了，不用再执行`merge`方法
* 数据量较小时使用插入排序法，此时需要改写插入排序法

### 自底向上归并排序

* 使用插入排序优化自底向上的归并排序
